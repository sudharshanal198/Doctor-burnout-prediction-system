🚀 Overview

The AI Doctor Burnout Prediction & Analysis System is a machine learning-powered web application designed to identify and analyze burnout risk among healthcare professionals.

It helps in early detection, workload analysis, and preventive recommendations, enabling hospitals to take proactive measures and improve doctor well-being.

🎯 Key Features
🔍 Burnout Risk Prediction using ML model
📊 Interactive Dashboard with real-time analytics
📈 Burnout Distribution Visualization (Charts)
📉 Trend Analysis (Weekly Monitoring)
🧠 AI-Based Recommendations for stress reduction
🧑‍⚕️ Doctor-wise Risk Assessment
